A simple and functional Text File Manager implemented in the C programming language.
This program allows users to perform common text-processing operations such as word counting, searching, replacing text, and counting characters/lines in a file.

 Features
  1. Word Frequency Counter

Counts how many times a specific word appears in the file.

 2. Word Finding Tool

Searches for a word and tells whether it exists in the file.

 3. Replace Word Tool

Replaces a given word with another word throughout the file and rewrites the updated content.

 4. Character & Line Count

Counts the total number of characters and lines present in the file.

 5. Dynamic File Reading

Reads the file into dynamically allocated memory.
